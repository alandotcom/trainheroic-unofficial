# Design system

## Reference

[agent-browser.dev](https://agent-browser.dev/) — simple docs, system fonts, Shiki-highlighted code blocks.

## Color strategy

Neutral only. White page, gray body text, `#f6f8fa` code blocks (GitHub light).

## Typography

System UI for everything. `ui-monospace` for code.

## Components

- **Code blocks**: Astro `Code` + Shiki `github-light`, copy button top-right
- **Lists**: normal bullets, no bordered rows
- **Inline code**: monospace only, no background
