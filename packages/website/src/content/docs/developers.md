Documentation for Claude Code, scripts, and custom apps. To connect Claude.ai in
the browser, use the [connector setup](../) on the home page.

- [Claude Code skill](../skill/): run `npx skills add`, then the agent calls
  `trainheroic`.
- [TypeScript SDK](../sdk/): package `@trainheroic-unofficial/js` for scripts
  and workers.
- [MCP reference](../mcp/): Cursor, Claude Desktop, local stdio servers, and
  the tool catalog.
